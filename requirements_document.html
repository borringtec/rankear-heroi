1. Objetivos

A Calculadora de Partidas Rankeadas é um sistema que visa classificar heróis de acordo com a quantidade de vitórias e derrotas, apresentando um nível correspondente.

1.1. Objetivos do Documento

Este documento tem como objetivo descrever e especificar os requisitos do sistema, servindo como guia para os desenvolvedores e interessados.

2. Escopo Geral do Produto

O sistema rankeia heróis com base em suas vitórias e derrotas, exibindo mensagens correspondentes ao nível alcançado.

3. Convenções, Termos e Abreviações

  • Nível – Classificação do herói com base nas vitórias e derrotas.

4. Requisitos

4.1. Requisitos Funcionais

RF001 Rankear o herói com base na quantidade de vitórias e derrotas.
Entrada Vitórias e derrotas do jogador.
Saída Nível do herói.
Pré-condição Informações de vitórias e derrotas fornecidas.
Pós-condição Nível do herói realizada com sucesso.

4.2. Requisitos Não Funcionais

RNF001 O sistema deve ser responsivo e funcionar em dispositivos móveis.

4.3. Requisitos de Domínio

RD001 Se vitórias < 10: Ferro

Se 11 ≤ vitórias ≤ 20: Bronze

Se 21 ≤ vitórias ≤ 50: Prata

Se 51 ≤ vitórias ≤ 80: Ouro

Se 81 ≤ vitórias ≤ 90: Diamante

Se 91 ≤ vitórias ≤ 100: Lendário

Se vitórias ≥ 101: Imortal
